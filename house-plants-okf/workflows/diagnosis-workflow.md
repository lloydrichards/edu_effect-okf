# Diagnosis Workflow

1. Name the symptom clearly.
2. Use [diagnostic triage](/problems/diagnostic-triage.md) to choose the next focused concept.
3. Check moisture, light, temperature, pests, and salts only as needed.
4. Apply the least disruptive fix first.
5. Judge success by new growth rather than old damaged tissue.

The workflow is intentionally narrow. The directory indexes carry broad navigation; focused diagnostic pages carry graph links.
